Плагины

Плагины - дополнения к серверу, которые предоставляют удобства игрокам.

  • - AdvancedAchievements - Дополнительные ачивки
  • - AdvancedBan - Плагин для банов
  • - BungeeTabListPlus - Таблист с поддержкой банжи
  • - BungeeTeleportManager - Плагин для телепортации с поддержкой банки
  • - ChatControlRed - Улучшенный чат и другие возможности для книг
  • - Chunkmaster - Предгенерация чанков
  • - CMI - Мульти инструмент, порталы, ванишь, голограммы и многое другое
  • - FastAsyncWorldEdit (WorldEdit) - Асинхронный WorldEdit
  • - HeadDatabase - База голов
  • - JPremium* - Плагин авторизации
  • - LogBlock - Логирование которое отслеживает изменения блоков
  • - LuckPerms - Плагин разрешений
  • - MysqlPlayerDataBridge - Синхронизация инвентарей игроков между серверами
  • - NoMobGriefing - Управление грифом отдельных мобов
  • - SkinsRestorer - Скины для не лицензионных игроков
  • - VillagerMarket - Магазины в игре
